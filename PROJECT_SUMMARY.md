## 🚀 Quick Start

### Option 1: View Locally (Immediate)
1. Navigate to Portfolio folder
2. Double-click `index.html`
3. Website opens in default browser

### Option 2: Deploy to GitHub Pages (Free, 10 minutes)
1. Create GitHub repository
2. Upload files
3. Enable GitHub Pages in settings
4. Website goes live at `https://username.github.io`

### Option 3: Deploy to Netlify (Free, 5 minutes)
1. Sign up at netlify.com
2. Connect GitHub
3. Select repository
4. Auto-deploys on push

See DEPLOYMENT.md for detailed instructions.
