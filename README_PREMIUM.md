## 🔧 Customization

### **Change Colors**
Edit `styles-v2.css`:
```css
:root {
    --primary: #YOUR_COLOR;
    --accent: #YOUR_ACCENT;
}
```

### **Change Font**
Edit `styles-v2.css`:
```css
body {
    font-family: 'Your Font', sans-serif;
}
```

### **Adjust Animations**
Edit `styles-v2.css`:
```css
--transition-fast: 150ms;  /* Faster */
--transition-base: 200ms;  /* Normal */
--transition-slow: 300ms;  /* Slower */
```

### **Modify Spacing**
Edit `styles-v2.css`:
```css
--spacing-lg: 1.5rem;  /* Increase for more space */
```
