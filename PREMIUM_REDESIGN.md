## 📋 Customization Guide

### **Change Primary Color**
Edit `styles-v2.css`:
```css
:root {
    --primary: #YOUR_COLOR;
    --primary-light: #YOUR_COLOR_LIGHT;
    --primary-dark: #YOUR_COLOR_DARK;
}
```

### **Modify Typography**
```css
:root {
    font-family: 'Your Font';
    --hero-title: 3.5rem; /* Adjust size */
}
```

### **Adjust Spacing**
```css
:root {
    --spacing-lg: 1.5rem; /* Increase for more space */
}
```

### **Customize Animations**
```css
--transition-base: 200ms cubic-bezier(...); /* Faster/slower */
```
